# Event Profit Planner

Калькулятор окупаемости мероприятия: выручка, расходы, прибыль, маржа, точка безубыточности и сценарии.

## Запуск локально

```bash
cd event-profit-planner
python3 -m http.server 8080
```

Откройте: `http://localhost:8080`

## Что считает

- ожидаемую выручку
- общие расходы
- ожидаемую прибыль
- маржу
- точку безубыточности (нужно продать билетов)
- 3 сценария: пессимистичный, базовый, оптимистичный
